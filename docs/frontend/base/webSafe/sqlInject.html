# SQL注入攻击

# SQL注入的危害

非法读取、篡改、删除数据库中的数据，盗取敏感信息，甚至注入木马等，危害极大。

# SQL注入的攻击方式

在查询参数里加入了SQL语句。

-- 这是一个简单的sql查询语句

select * from blogs where name='zhangsan' and keyword like '%guan%';
